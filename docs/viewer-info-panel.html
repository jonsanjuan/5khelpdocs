File Info Panel

The File Information tab displays details about the File itself, such as the file name, file size, and when it was uploaded. Additionally, 5th Kind extracts any metadata that is in the File when you upload it. That metadata will be diplayed here as well. To access the File Information tab, click the icon in the Viewer.
